Compilable Compact smart contract examples from 8 repositories — beginner contracts, reusable modules, token implementations, and full applications with witnesses and tests. All examples compile with `pragma language_version >= 0.22` and support full proof generation.

## Skills

### compact-examples:code-examples

Comprehensive library of working Compact smart contract examples organized into five categories:

- **Getting Started** — Simple contracts like Counter and Bulletin Board to learn Compact fundamentals
- **Modules** — Reusable components including Ownable, AccessControl, Pausable, and Initializable patterns
- **Tokens** — Token implementations (FungibleToken, NonFungibleToken, MultiToken) and token-based applications
- **Privacy & Cryptography** — Examples using Schnorr signatures, zero-knowledge proofs, and shielded operations
- **Applications** — Complete applications including Kitties, ZKLoan, RWA, and TBTC implementations

All contracts are fully compilable and tested with witness and proof generation included.
